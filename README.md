# How to use Git and GitHub:
Data practices workshop
CSUN BEER club, Spring 2026

## 1. Get Git
**Mac (or Linux)** -
Good news: you already have Git installed! No need to download anything.

**Windows** -
Downnload Git from the Git website: https://git-scm.com/install/windows

## 2. Make a GitHub account
Create an account or login at https://github.com/.
You can use your CSUN email if you want, just make sure to link your personal email as a backup in your account settings to avoid being locked out when you graduate.
